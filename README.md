# east-mas-ui

邮件归档系统 UI

### 项目文件目录

```
|-- .gitignore ------------git屏蔽规则
|-- README.md -------------自述文件
|-- index.html ------------入口文件
|-- package.json ------------依赖配置文件
|-- vite.config.js ------------vite 配置文件
|-- dist ------------打包后输出文件
|-- doc ------------文档位置
|-- public ------------项目非打包文件目录
|-- src ------------源码目录
| |-- App.vue ------------跟组件
| |-- main.js ------------入口js文件
| |-- api ------------接口文件位置
| |-- assets ------------打包静态资源文件夹
| |-- components ------------组件目录
| |-- router ------------项目路由
| |-- style ------------项目样式文件
| |-- tools ------------项目自定义工具库
| |-- utils ------------自定义工具库
| | |-- http.js ------------接口请求拦截器
| |-- views ------------所有视图组件
|-- viteCacheDir ------------vite运行缓存

```
